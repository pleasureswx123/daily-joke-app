# 技术上下文

## 开发环境
- Node.js >= 18.0.0
- npm >= 9.0.0

## 核心技术栈
- React 18
- TypeScript 5
- TailwindCSS 3
- Next.js 14 (可选，用于开发环境)

## 依赖管理
```json
{
  "dependencies": {
    "react": "^18.2.0",
    "react-dom": "^18.2.0",
    "tailwindcss": "^3.3.0",

  },
  "devDependencies": {
    "typescript": "^5.0.0",

    "@types/react-dom": "^18.2.0"
  }

```

## 代码规范
- 使用ESLint进行代码检查
- 使用Prettier进行代码格式化
- 遵循React最佳实践和TypeScript严格模式
